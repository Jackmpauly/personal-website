Vela

A Python framework for Gemini servers.

Usage

A minimal example:
pip install vela # Generate an SSL certificate and private key. Using OpenSSL: openssl req -x509 -newkey rsa:4096 -out cert.pem -keyout key.pem -days 365 -nodes -subj '/CN=localhost'
from vela import Server app = Server(cert='cert.pem', key='key.pem') @app.route('/') def hello(req, res): res.send_file('hello.gmi') PORT = 5000 print(f'Server is listening on port {PORT}...') app.listen(PORT)
